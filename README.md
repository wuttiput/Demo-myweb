# Personal Hub — Tech News Reader & Multi-Model AI Chat

**Personal Hub** เป็นเว็บบอร์ดศูนย์รวมเครื่องมือส่วนตัวที่ออกแบบมาด้วยอินเตอร์เฟสแบบ **Premium Dark Theme & Glassmorphism** ทันสมัย สวยงาม และใช้งานง่าย พัฒนาด้วย Flask (Python) และ SQLite เป็นฐานข้อมูลหลัก

โปรเจกต์นี้รวมสองฟีเจอร์เด่น: **ระบบอ่านข่าวเทคโนโลยี & AI แปลไทย** และ **ระบบแชทอัจฉริยะหลายค่าย (Gemini, DeepSeek, Qwen)** พร้อมเครื่องมือการจัดการชีวิตประจำวัน (ระบบรายรับ-รายจ่าย และระบบบันทึกสิ่งที่ต้องทำ)

---

## 🌟 ฟีเจอร์หลัก (Key Features)

### 1. 🤖 Multi-Model AI Chat (แชทบอร์ดอัจฉริยะหลายค่าย)
* **การใช้งานแบบ Multi-Model**: สามารถสลับคุยกับ AI ของแต่ละค่ายได้ทันทีผ่าน Dropdown (Gemini, DeepSeek, Qwen)
* **สิทธิ์การเข้าถึง**:
  * **Gemini 2.5 Flash**: ใช้งานฟรีผ่าน Google AI Studio API Key (มีความเสถียรสูงและฟรี)
  * **DeepSeek Chat (Official API)**: รองรับการต่อคีย์ตรงเพื่อความเร็วและความแม่นยำสูงสุด
  * **OpenRouter Free Models**: เชื่อมต่อโมเดลรุ่นฟรีของ DeepSeek และ Qwen ผ่าน OpenRouter
* **การจัดการประวัติการสนทนา**:
  * บันทึกประวัติการแชทลง SQLite ในเครื่องอย่างปลอดภัย
  * ระบบสร้างห้องแชทใหม่ / ลบแชทเก่า (ChatGPT-Style)
  * เปลี่ยนชื่อหัวข้อห้องสนทนาอัตโนมัติจากคำถามแรกที่พิมพ์ส่ง
* **UI/UX พรีเมียม**: ส่งข้อความแบบ Dynamic AJAX (ไม่ต้องรีเฟรชหน้าจอ) พร้อมอนิเมชั่นพิมพ์ตอบ (Typing Indicator)

### 📰 2. Tech & AI News Reader (ระบบอ่านและสรุปข่าวไอที)
* **ดึงข่าวสารอัตโนมัติ**: ดึงข้อมูลอัปเดตจาก RSS Feeds ของสำนักข่าวเทคโนโลยีชั้นนำระดับโลก
* **แปลและสรุปด้วย AI**: ใช้โมเดล `gemini-2.5-flash` ในการแปลหัวข้อข่าวและเขียนสรุปใจความสำคัญเป็นภาษาไทยโดยอัตโนมัติ
* **ประหยัดทรัพยากร**: ระบบจะอัปเดตข่าวใหม่ทุกๆ 6 ชั่วโมง และมีการจัดเก็บแคชลงในฐานข้อมูล เพื่อป้องกันการเรียกใช้ API เกินขีดจำกัด (Rate Limit Friendly)
* **ระบบกรองคำหลัก (Fallback)**: หาก API Key มีปัญหา ระบบจะกรองข่าวภาษาอังกฤษอัตโนมัติมาแสดงผลเพื่อให้ใช้งานต่อได้ไม่สะดุด

### 💼 3. Finance Tracker (ระบบบันทึกรายรับ-รายจ่าย)
* บันทึกรายการรายรับและรายจ่าย พร้อมระบุประเภท วันที่ และคำอธิบาย
* แดชบอร์ดสรุปยอดรวมรายรับ ยอดรวมรายจ่าย และเงินคงเหลือปัจจุบันแบบเรียลไทม์

### 📝 4. Todo List (ระบบจัดการสิ่งที่ต้องทำ)
* เพิ่ม แก้ไขสถานะการทำงาน (ทำเสร็จแล้ว/ยังไม่ได้ทำ) และลบรายการสิ่งที่จะทำในแต่ละวัน

### 🔒 5. User Authentication (ระบบสมาชิกและระบบรักษาความปลอดภัย)
* สมัครสมาชิกและเข้าสู่ระบบแยกบัญชีส่วนตัว
* เข้ารหัสรหัสผ่านของผู้ใช้ (Password Hashing) ก่อนบันทึกเข้า SQLite เพื่อความปลอดภัยสูงสุด

## 🛠️ วิธีการติดตั้งและรันใช้งาน (Installation & Setup)

### 1. ติดตั้ง Python
ตรวจสอบให้แน่ใจว่าเครื่องคอมพิวเตอร์ของคุณติดตั้ง Python เวอร์ชัน 3.8 ขึ้นไปเรียบร้อยแล้ว

### 2. โคลนคลังโค้ดนี้ลงเครื่องของคุณ
```bash
git clone https://github.com/wuttiput/Demo-myweb.git
cd Demo-myweb
```

### 3. ติดตั้ง Packages ที่ต้องใช้
แนะนำให้รันคำสั่งติดตั้งผ่าน Terminal ในโฟลเดอร์ของโปรเจกต์:
```bash
pip install -r requirements.txt
```

### 4. การตั้งค่าสิทธิ์เข้าใช้งาน API (.env)
คัดลอกไฟล์ `.env.example` และเปลี่ยนชื่อเป็น `.env` จากนั้นใส่ API Keys ของคุณ:
```env
# Google AI Studio API Key (ฟรี) -> ได้จาก https://aistudio.google.com/
GEMINI_API_KEY=your_gemini_api_key_here

# OpenRouter API Key (สำหรับโมเดลฟรีของ DeepSeek/Qwen) -> https://openrouter.ai/
OPENROUTER_API_KEY=your_openrouter_api_key_here

# DeepSeek Official API Key (ต่อตรง/เติมเงินเพื่อความเร็วและเสถียร 100%) -> https://platform.deepseek.com/
DEEPSEEK_API_KEY=your_deepseek_api_key_here
```

### 5. เปิดทำงานระบบ (Run Server)
รันแอปพลิเคชันด้วยคำสั่ง:
```bash
python app.py
```
จากนั้นเปิดบราวเซอร์และเข้าไปที่ลิงก์: **`http://127.0.0.1:5000/`**
